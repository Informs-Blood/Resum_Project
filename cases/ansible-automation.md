# Кейс: Автоматизация процессов с Ansible

## Задача

Автоматизировать рутинные задачи для ускорения развёртывания инфраструктуры и настройки серверов.

## Решение

- Создание **playbooks** для автоматической настройки серверов (установка ПО, конфигурация сетевых сервисов, брандмауэры)
- Разработка **ролей Ansible** для повторного использования
- Интеграция Ansible с **CI/CD-конвейерами** для автоматического развёртывания приложений

## Инструменты

Ansible, Git, CI/CD

## Результат

- Сокращение времени на развёртывание новых серверов на **~80%**
- Минимизация ошибок за счёт автоматизации рутинных задач
