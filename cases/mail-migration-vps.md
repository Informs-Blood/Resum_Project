# Кейс: Миграция почтовых серверов на собственные VPS

## Задача

Перевести почту с облачного провайдера на собственные VPS, снизив затраты.

## Решение

- Настройка почтового сервера на Linux: **Postfix**, **Dovecot**
- Перенос учётных записей и данных пользователей без потерь
- Полный цикл работ: проектирование, развёртывание, миграция

## Инструменты

Linux, Postfix, Dovecot, VPS

## Результат

- Снижение затрат на почту на **~30%**
- Успешная миграция без потери данных
