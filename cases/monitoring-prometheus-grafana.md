# Кейс: Внедрение системы мониторинга Prometheus + Grafana + Alertmanager + Pushgateway

## Задача

Отсутствовал централизованный мониторинг и оперативное оповещение о сбоях инфраструктуры. Требовалось повысить прозрачность и сократить время реакции на инциденты.

## Решение

- Развёртывание **Prometheus** для сбора метрик с серверов, приложений и сетевого оборудования
- Настройка **Grafana** для визуализации и дашбордов
- Интеграция **Alertmanager** для оповещений через Telegram API и систему управления инцидентами
- Использование **Pushgateway** для сбора метрик из batch-задач и скриптов

## Инструменты

Prometheus, Grafana, Alertmanager, Pushgateway, Telegram API

## Результат

- Сокращение времени реакции на инциденты на **~70%** за счёт автоматических оповещений
- Повышение прозрачности инфраструктуры благодаря детальным дашбордам
