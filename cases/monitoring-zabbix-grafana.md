# Кейс: Внедрение Zabbix + Grafana для мониторинга

## Задача

Организовать централизованный мониторинг сети и серверов для быстрого обнаружения инцидентов.

## Решение

- Развёртывание **Zabbix 6.2** для сбора метрик с серверов и сетевого оборудования
- Интеграция **Grafana** для визуализации и дашбордов
- Настройка оповещений

## Инструменты

Zabbix, Grafana

## Результат

- Сокращение времени реакции на инциденты на **~50%**
- Прозрачная картина состояния инфраструктуры
