# Кейс: План аварийного восстановления (Disaster Recovery Plan)

## Задача

Создать план аварийного восстановления для минимизации простоев при сбоях или катастрофах.

## Решение

- Разработка DRP с учётом критических систем
- Настройка регулярного резервного копирования с **Veeam Backup & Replication**
- Создание тестового стенда для проверки плана восстановления
- Интеграция DRP с **Prometheus** для автоматического запуска восстановления при сбоях

## Инструменты

Veeam Backup & Replication, Prometheus

## Результат

- Время восстановления критических систем сократилось до **~2 часов**
- Уменьшение финансовых потерь за счёт своевременного восстановления данных
