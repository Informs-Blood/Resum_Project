# Кейс: Миграция баз данных на PostgreSQL с высокой доступностью

## Задача

Перевести базы данных с MySQL на PostgreSQL для повышения производительности и отказоустойчивости.

## Решение

- Миграция данных с минимальным временем простоя
- Настройка **репликации PostgreSQL** для высокой доступности
- Интеграция с **Prometheus** для мониторинга состояния БД
- Создание документации для команды разработчиков

## Инструменты

PostgreSQL, Prometheus

## Результат

- Увеличение производительности БД на **~40%**
- Снижение простоев за счёт репликации и автоматического переключения
